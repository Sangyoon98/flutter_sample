# Flutter Instagram Clone Study Docs

이 폴더는 에이전트가 학습 진행 상황을 잊지 않고 이어가기 위한 문서 공간이다.
코드 변경이 있는 학습 세션이 끝날 때마다 관련 문서를 반드시 갱신한다.

## 문서 목록

- `LEARNING.md`: 현재 활성 학습 트랙 색인
- `LEARNING_FLUTTER_INSTAGRAM_CLONE.md`: 공식 문서 기반 Fluttergram 활성 학습 플랜
- `clone_target.md`: Stitch UI와 CATAAS API 기준 구현 목표
- `study_plan.md`: 인스타그램 클론을 만들며 배울 Flutter/Dart 전체 학습 계획
- `progress_log.md`: 날짜별 학습 기록, 구현 범위, 다음 작업
- `session_protocol.md`: 매번 학습을 진행할 때 따르는 운영 규칙
- `learning_notes.md`: 공부 중 누적하는 핵심 개념 정리

## 기본 학습 방식

1. 사용자가 "오늘 공부", "다음 작업", "이어 하자"처럼 요청하면 `LEARNING.md`를 먼저 확인한다.
2. 활성 트랙 문서의 첫 미완료 단계에 맞춰 작은 목표 하나를 정한다.
3. 사용자가 직접 따라 칠 수 있도록 코드와 설명을 함께 제공한다.
4. 앱 코드 변경은 사용자가 직접 작성하는 것을 기본으로 한다.
5. 세션 종료 시 활성 트랙 문서의 진행 기록, 호기심 기록, 실습 검증 기록을 갱신한다.
6. 코드와 문서가 함께 정리된 뒤 커밋한다.

## 현재 프로젝트 상태

- 프로젝트 성격: Flutter 학습용 인스타그램 클론
- 초기 상태: `flutter create`로 생성된 기본 카운터 앱
- 현재 앱 상태: Fluttergram 첫 홈 화면 뼈대 작성
- 현재 활성 학습 트랙: `FLUTTER_INSTAGRAM_CLONE`
- 현재 학습 단계: `LEARNING_FLUTTER_INSTAGRAM_CLONE.md`의 첫 미완료 단계
