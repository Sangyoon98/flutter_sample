# Learning Tracks

이 파일은 학습 트랙 색인이다.
현재 활성 트랙은 `FLUTTER_INSTAGRAM_CLONE` 하나다.

## 현재 학습 트랙

| 트랙 ID | 주제 | 상태 | 활성 문서 | 메모 |
|---|---|---|---|---|
| `FLUTTER_INSTAGRAM_CLONE` | Flutter 공식 문서 기반 인스타그램 UI 클론 | 진행 중 | `LEARNING_FLUTTER_INSTAGRAM_CLONE.md` | Stitch 홈 피드/Explore UI와 CATAAS 고양이 이미지 API를 사용한다. |

## 트리거

아래 요청은 활성 트랙의 첫 미완료 단계를 이어간다.

- "학습 시작"
- "학습시작"
- "오늘 공부"
- "다음 단계"
- "이어 하자"

## 운영 원칙

- 진행 상태는 활성 트랙 문서에 기록한다.
- 앱 코드는 사용자가 직접 따라 치는 것을 기본으로 한다.
- 에이전트는 사용자가 명시적으로 요청하지 않으면 앱 코드, 테스트 코드, 빌드 파일을 직접 수정하지 않는다.
- 복습 문서는 `LEARNING_REVIEW_FLUTTER_INSTAGRAM_CLONE_STEP_{단계번호}.md` 형식으로 만든다.
