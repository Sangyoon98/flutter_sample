# 3단계 복습: 홈 피드 정적 UI 구성

## 학습 출처

- Flutter layouts: https://docs.flutter.dev/ui/layout
- Flutter widgets: https://docs.flutter.dev/ui/widgets

## 핵심 요약

- `Column`은 자식 위젯을 세로로 배치한다.
- `Row`는 자식 위젯을 가로로 배치한다.
- 포스트 카드 전체는 헤더, 이미지, 액션, 좋아요, 캡션이 세로로 쌓이므로 `Column`을 사용한다.
- 포스트 헤더와 액션 버튼 영역은 한 줄에 배치하므로 `Row`를 사용한다.
- `AspectRatio(aspectRatio: 1)`은 영역을 1:1 정사각형 비율로 유지한다.
- `Spacer`는 `Row`나 `Column` 안의 남는 공간을 차지해서 다른 위젯을 끝으로 밀어낼 때 사용한다.

## 사용자 질문 기반 정리

### 포스트 카드 방향

**질문:** 인스타그램 포스트 카드 전체를 세로로 쌓으려면 무엇을 쓰는가?

**답:** `Column`을 쓴다. 포스트 카드 전체는 헤더, 이미지, 액션 row, 좋아요 수, 캡션이 위에서 아래로 배치되는 세로 구조이기 때문이다.

### 포스트 헤더 방향

**질문:** 프로필 아바타, 유저명, 더보기 버튼을 한 줄에 놓으려면 무엇을 쓰는가?

**답:** `Row`를 쓴다. 세 요소가 좌우 방향으로 나란히 배치되어야 하기 때문이다. 더보기 버튼을 오른쪽 끝으로 보내기 위해 중간에 `Spacer`를 둘 수 있다.

### 이미지 비율

**질문:** 게시글 이미지 영역을 정사각형으로 유지하려면 무엇을 쓰는가?

**답:** `AspectRatio`를 쓰고 `aspectRatio: 1`로 설정한다. 이렇게 하면 부모가 주는 너비 안에서 가로와 세로 비율을 1:1로 유지한다.

## 헷갈리기 쉬운 구분

- `Column`은 세로 배치, `Row`는 가로 배치다.
- `Padding`은 자식 위젯 주변의 안쪽 여백을 만든다.
- `SizedBox`는 고정 크기 또는 빈 간격을 만들 때 쓴다.
- `AspectRatio`는 실제 이미지를 불러오는 위젯이 아니라 영역의 비율을 잡는 위젯이다.
- `CircleAvatar`는 원형 아바타 UI를 빠르게 만들 때 쓴다.

## 한 줄 복습

포스트 카드 전체는 `Column`, 한 줄짜리 헤더와 버튼 영역은 `Row`, 정사각형 이미지 영역은 `AspectRatio(aspectRatio: 1)`로 만든다.

## 다시 물어볼 질문

1. 포스트 카드 전체에는 왜 `Column`을 쓰는가?
2. `Row` 안에서 버튼을 오른쪽 끝으로 밀려면 어떤 위젯을 쓸 수 있는가?
3. `AspectRatio(aspectRatio: 1)`은 어떤 비율을 의미하는가?
